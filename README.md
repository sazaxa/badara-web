# SAZAXA SHIPMENT-WEB

## sazaxa shipment Fornt-End Source.

#### Git Clone.

```
git clone https://github.com/sazaxa/shipment-web.git
```

#### Config
```
cd ${project Root}
vi .env
```

#### Config local.env

```
REACT_APP_ENV=local
REACT_APP_API_URL=http://localhost:8080
```

#### Config prod.env

```
REACT_APP_ENV=prod
REACT_APP_API_URL=https://api.badara.kr
```

#### Node Module Install.

```
yarn install
```

### Local Develper

```
yarn start
```

### Prod Build

```
yarn build:prod
```
